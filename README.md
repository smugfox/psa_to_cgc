# PSA → CGC Crossover Reference

Personal reference library of Pokémon cards that crossed over from PSA to CGC Pristine 10.
Used as visual reference when hunting for undergraded PSA copies on eBay.

## Cards

| Card | PSA | CGC | Folder |
|---|---|---|---|
| Giovanni's Persian — Gym Challenge #8/132, Holo 1st Ed. (2000) | 8 | **Pristine 10** (top pop) | [cards/2000-gym-challenge-08-giovannis-persian-1st-ed](cards/2000-gym-challenge-08-giovannis-persian-1st-ed/card.md) |

## Structure

Each card gets a folder under `cards/` named `<year>-<set>-<number>-<name>`:
- `card.md` — cert links, grades, pop data, sales comps
- `psa-<cert>-front.jpg` / `psa-<cert>-back.jpg` — PSA slab scans (large variant from PSA CDN)
- `cgc-<cert>-front.jpg` / `cgc-<cert>-back.jpg` — CGC slab scans (full-res OBV/REV from CGC S3)
